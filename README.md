### SDK集成

# > 方式一[ ![Download](https://api.bintray.com/packages/dyncanyrtc/ar_dev/board/images/download.svg) ](https://bintray.com/dyncanyrtc/ar_dev/board/_latestVersion)
添加Jcenter仓库 Gradle依赖：

```
dependencies {
  compile 'org.ar:board:3.0.0'
}
```
